# What is CMP?

CMP (Cloud Management Platform) is a multi-cloud orchestration platform that enables cloud service providers to offer Infrastructure-as-a-Service (IaaS) to their customers. It abstracts the complexity of underlying hypervisors and cloud orchestrators behind a unified interface for both admins and end users.

## Key capabilities

* Unified customer portal for VM, storage, networking, and more
* Support for multiple orchestrators from a single admin panel
* Flexible billing: hourly (pay-as-you-go), monthly, quarterly, and yearly
* Prepaid and postpaid account models
* Quota management at global, account, and project levels
* SSO and 2FA via Keycloak
* White-label branding and customization

## Who is this documentation for?

This documentation is intended for **CMP administrators** — cloud providers who deploy and manage CMP on behalf of their customers. End-user guides are separate.
